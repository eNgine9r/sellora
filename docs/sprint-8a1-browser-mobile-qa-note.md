# Sprint 8A.1 Browser/Mobile QA Temporary Runner

This branch is QA-only and must not be merged.

It runs Playwright against staging for:

- 1366x768
- 375x812
- 390x844
- 430x932
- 768x1024

Covered routes:

- Login
- Dashboard
- Workspace switch safety
- Leads
- Customers
- Products
- Inventory
- Orders
- Shipments
- Finance
- Advertising
- Analytics
- Settings
- Team
- Logout

Evidence is sanitized. Credentials, tokens, Authorization headers, passwords and API keys must not appear in logs or artifacts.
